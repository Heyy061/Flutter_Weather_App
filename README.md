# weather_app

Flutter Weather App:-

A simple weather application built using Flutter and Dart. The app fetches real-time weather data from a weather API and displays current weather conditions in a clean, user-friendly interface.

Features:-
•	🌤️ Current weather information
•	🌡️ Temperature display
•	💧 Humidity
•	🌬️ Wind speed
•	☁️ Weather conditions

Tech Stack:-

•	Flutter
•	Dart
•	Weather API
•	HTTP package


Learning Note
This project was initially built by following a Flutter YouTube tutorial to learn API integration and Flutter development. I am continuing to improve it by adding new features and enhancing the user interface as I learn.
